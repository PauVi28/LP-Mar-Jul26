# ForgottenTale - Team 11 - Santi Celi, Yazid Armijos, Camilo Arias

Juego de combate por turnos desarrollado en **Python con Pygame** para la materia de Lenguajes de Programación (UIDE).

## Cómo ejecutar el juego

1. Asegúrate de tener **Python 3** instalado.
2. Instala la librería Pygame con:
   pip install pygame

- Librería Pygame: Utilizada para la creación de la ventana, manejo de eventos, dibujado y control del juego.
- Licencia: LGPL
- Sitio oficial: https://www.pygame.org/
- Inspiración:
    - El sistema de combate por turnos y la navegación con teclado está inspirado en juegos como Undertale, Mario & Luigi: Bowser's Inside History.
