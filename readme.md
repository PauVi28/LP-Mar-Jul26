Yo, Matheo Naranjo, declaro que el presente proyecto fue desarrollado con fines académicos para la asignatura correspondiente al desarrollo de software y redes LAN.

El código principal del sistema, la lógica del juego y la implementación cliente-servidor fueron desarrollados por el autor del proyecto. Además, se utilizaron herramientas y recursos externos únicamente como apoyo educativo y de aprendizaje.

### Librerías Utilizadas

* `socket` → Comunicación cliente-servidor en red LAN.
* `threading` → Manejo de múltiples conexiones.
* `pygame` (si aplica) → Desarrollo de interfaz gráfica y elementos visuales del juego.
* Librerías estándar de Python.
